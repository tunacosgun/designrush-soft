## 🚀 Installation & Usage

Because this project is built gracefully with pure frontend web technologies, absolutely no complex build steps or dependencies are needed to view it.

1. **Clone the repository:**
   ```bash
   git clone https://github.com/tunacosgun/designrush-soft.git
   cd designrush-soft
   ```
2. **Launch locally:**
   Simply double-click the `index.html` file to open it in your default web browser, or serve it using any local dev server:
   ```bash
   # using python
   python -m http.server 8000
   
   # using npx
   npx serve .
   ```
